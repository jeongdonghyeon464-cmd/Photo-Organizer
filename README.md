# Photo Organizer

Automatically organize photos and videos by Year / Month.

---

## Features

- JPG
- HEIC
- PNG
- DNG
- MOV
- MP4

- EXIF Priority
- Creation Date Fallback
- Duplicate Skip

- Folder Analysis
- GUI
- Progress Bar
